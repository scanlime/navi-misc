xchat-gnome Building from SVN

We highly recommend installing libsexy from Christian Hammond's SVN for, well, extra sexyness. If you're on pango 1.10.0, this will give you problems -- upgrade to 1.10.1 or later

DO NOT expect support from the normal X-Chat developers, and do not submit bug reports/feature requests/etc to them.

I've recently added a convenience script for myself. If you don't need any fancy configure options and don't mind building with a prefix of /usr, you can just run this. Keep in mind it doesn't actually run make install, just builds.

svn cat http://svn.navi.cx/misc/trunk/xchat-gnome/checkout-build.sh | sh

Otherwise, do this:

$ cvs -d :pserver:anonymous@cvs.sf.net:/cvsroot/xchat co xchat2
$ cd xchat2
xchat2$ svn cat http://svn.navi.cx/misc/trunk/xchat-gnome/buildsystem.diff | patch -p0
xchat2$ svn co http://svn.navi.cx/misc/trunk/xchat-gnome/plugins
xchat2$ pushd src
src$ svn co http://svn.navi.cx/misc/trunk/xchat-gnome/fe-gnome/
src$ popd
xchat2$ ./autogen.sh
xchat2$ ./configure --enable-maintainer-mode
(make sure that gnome support was found)
xchat2$ make
xchat2$ cd src/fe-gnome
fe-gnome$ gconftool-2 --install-schema-file=apps_xchat.schemas
fe-gnome$ ./xchat-gnome
