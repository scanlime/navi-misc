Welcome!

Welcome to the Firestarter project homepage. We aim to provide the quality game experience you're used to in the regular version of BZFlag. We also plan on adding:

  • New styles of gameplay:
    • Deathmatch (known as FFA)
    • Capture the Flag
    • Rabbit Hunt
    • Assault
    • Bombing Run
    • More
  • Better Graphics
  • Improved Input Capabilities
  • Tighter, cleaner codebase
  • Integration with:
    • SDL for input;
    • Ogre for 3D graphics;
    • eNet for networking,
    • More

Please check out the links to the left to get further information.
