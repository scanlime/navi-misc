xchat-gnome Building from SVN

Because xchat-gnome is maintained as a patch-set to xchat and things live in separate repositories, building it from subversion/cvs is an involved process. Note that we require GTK+ 2.4, but you'll get better functionality if you're using GTK+ 2.5.

DO NOT expect support from the normal X-Chat developers, and do not submit bug reports/feature requests/etc to them.

$ cvs -d :pserver:anonymous@cvs.sf.net:/cvsroot/xchat co xchat2
$ cd xchat2
xchat2$ svn cat http://navi.cx/svn/misc/trunk/xchat-gnome/buildsystem.diff | patch -p0
xchat2$ svn co http://navi.cx/svn/misc/trunk/xchat-gnome/plugins
xchat2$ pushd src
src$ svn co http://navi.cx/svn/misc/trunk/xchat-gnome/fe-gnome/
src$ popd
xchat2$ ./autogen.sh
xchat2$ ./configure --enable-maintainer-mode
(make sure that gnome support was found)
xchat2$ make
xchat2$ cd src/fe-gnome
fe-gnome$ gconftool-2 --install-schema-file=apps_xchat.schemas
fe-gnome$ ./xchat-gnome
