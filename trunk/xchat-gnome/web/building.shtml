xchat-gnome Building from SVN

We highly recommend installing libsexy for extra sexyness. Make sure you use 0.1.4 or later, and a pango other than 1.10.0 (earlier or later is fine).

DO NOT expect support from the normal X-Chat developers, and do not submit bug reports/feature requests/etc to them.

Building xchat-gnome is now very much like building any other gnome application.

$ svn co http://svn.navi.cx/misc/trunk/xchat-gnome
$ cd xchat-gnome
xchat-gnome$ ./autogen.sh
xchat-gnome$ make
xchat-gnome$ sudo make install
