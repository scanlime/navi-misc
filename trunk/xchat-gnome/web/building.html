The first requirement is to fetch X-Chat CVS. You can do this from their sourceforge project page at SourceForge. Make sure to grab the 'xchat2' module. Check the buildsystem patch out from subversion and apply it to the xchat trunk. Then check out the gnome frontend into xchat2/src. After this, the process to build consists of:

xchat2$ ./autogen.sh
xchat2$ ./configure --enable-maintainer-mode --enable-xft --enable-gnomefe
(make sure that gnome support was found, and gtkspell if desired)
xchat2$ make
xchat2$ cd src/fe-gnome
fe-gnome$ ./xchat-gnome
