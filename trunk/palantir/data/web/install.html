Requirements
1. >= Python 2.2.3 (I use Python 2.3.3 but 2.2.3 should work)
2. pyGTK and glade
3. pyxml (This will hopefully be replaced soon to cut down on the number of requirements)
4. twisted modules
Instructions - Linux
Right now it's not terribly trivial to install Palantir, you'll need to have a copy of subversion to get the modules.

1. Make sure you have the above requirements installed on you computer.
2. $ svn co http://navi.cx/svn/misc/trunk/palantir
3. $ cd palantir

At this point you can stop and just run palantir-client.py from the directory you checked out. The next steps should install the modules into your site-packages dir and the scripts into /usr/bin/. But the setup.py script is mostly untested it was working and then I decided to make some changes to where it installed things and I haven't tested it yet. Everything should run ok after step 3.

4. $ ./setup.py install **NOTE: You'll need root priveleges to do this for now
5. $ palantir-client.py

Because this is the subversion repository and under development there are no guarantees that everything will work. This project is not quite yet ready for release, but help with testing and the like is appreciated. Patches are welcome, see the TODO.
