What Is Palantir?

Palantir is a small IRC client designed for playing RPGs (role playing games) over IRC. It was inspired by the OpenRPG project and hopes to take the ideas behind that project to the next level. Palantir has a couple of RPG specific features such as built-in character sheets and dice rolling. Palantir can connect to any IRC server and use any of its RPG features because these have all been implemented utilising the existing IRC protocols. The hope is that this will make it easier and more convenient for people to use.
